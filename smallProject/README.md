# Small Project Boiler Plate

## About
This is the starting point that I use when I build a new project.

## Technologies
* Stylus
* Grunt
* NPM

## Getting Started
1. Install grunt dependencies
	
	```
	npm install
	```
2. Update package.json
	* Update your Name, Description, Repository, Homepage etc.
3. Run Grunt
	
	``` 
	grunt
	```
